<h1></h1> 제목지정 태그
<p></p> 문단지정 태그 
**태그의 포함관계 : 엘리먼트안에 또다른 엘리먼트를 작성하기위한 관계

html 관계 
html : hyper text markup language (컨텐츠 구조 언어)
태그간의 상속관계가 존재
태그간의 시블링즈 관계가 존재(형제)
=> html trees로 최종적으로 구성된다.
<div></div> //부모역할을 하는 태그
포함관계 구성시 부모자식관계는 지속적으로 유지한다.
*하나의 부모요소에 여러개의 형제태그가 존재할수있다.

html 속성 
태그속성 : 특별한 설정값을 주고싶을때 사용한다.
ex) <태그속성>콘텐츠</태그>
<a></a> 하이퍼 링크 
<a href="연결주소">콘텐츠</a> 하이퍼링크의 속성 
empty 엘리먼트 : 쌍을 이루지않는 태그 
ex) <br> : 줄바꿈 
<img> 태그 : 이미지 태그 
src = "이미지 주소" 의 속성을 가진다.
id 속성 : 유일한 요소에 대한 식별자 
class 속성 : 다중요소에 대한 식별자
